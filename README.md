# Speechie Library

This project was designed for use by speech pathologists and treats resource searching and storage in an effective manner. By reducing clutter and the strenous use of file explorer, this AI powered chat engine it able to search for resources with a few words and stores them alongisde with a list of patients which can be linked to each resource. This has been full deployed on render and uses the MERN stack.
